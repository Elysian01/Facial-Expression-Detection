# Facial-Expression-Detection

### Detects Face using OpenCv and then Classify its into Emotion

<img src = "./videos/Working_Demo_GIF.gif" width = 500px height=300px alt = "#"/>

### The Model can classify 7 different Emotion

* Angry
* Disgust
* Fear
* Happy
* Sad
* Surprise
* Neutral
